# Text Algorithms 2024/2025

Repozytorium zawiera implementacje kluczowych algorytmów tekstowych przygotowanych w ramach zajęć laboratoryjnych.

## Zawartość

1. **[Regex & Text Analysis](./lab1)**
   * Zbiór skryptów Pythonowych do ekstrakcji danych, analizy statystycznej tekstu oraz konwersji wyrażeń regularnych na deterministyczne automaty skończone.
2. **[String Matching Algorithms](./lab2)**
   * Kompleksowa implementacja kluczowych algorytmów wyszukiwania wzorca w tekście, obejmująca podejścia od naiwnych po zaawansowane techniki oparte na haszowaniu, tablicach preprocesingu i automatach.
3. **[Advanced String Algorithms & Approximate Matching](./lab3)**
   * Zaawansowane przetwarzanie tekstów, wyszukiwanie wielu wzorców naraz, szukanie przybliżone oraz struktury 2D.
4. **[Suffix Structures & Sequence Analysis](./lab4)**
   * Drzewa Sufiksowe oraz Tablice Sufiksowe.
5. **[Text-similarity-algorithms](./lab4)**
   * Implementacje algorytmów służących do porównywania dwóch ciągów znaków, wyznaczania kosztu ich edycji oraz znajdowania optymalnego wyrównania jednego tekstu na   drugi.
